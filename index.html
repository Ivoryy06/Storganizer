About Storganizer

Generated by Storganizer. This tool identifies true duplicates and visually similar media using perceptual pHash and dHash analysis.

How to Use Storganizer

Storganizer finds duplicate and near-duplicate media files using perceptual hashing.

Download & Run

# Build from source
go build -o storganizer .

# Scan current directory
./storganizer

# Scan specific folder
./storganizer -path ~/Pictures

# Generate HTML report
./storganizer -path ~/Pictures -html
# Build from source
go build -o storganizer.exe .

# Scan disk path
.\storganizer.exe -path D:\Photos

# Generate HTML report
.\storganizer.exe -path D:\Photos -html

Flags

-path        Directory to scan (default: .)
-threshold   Perceptual distance (default: 10)
-json        Output JSON file
-html        Output HTML report
